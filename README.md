# ✈️ Crew Scheduling Mini Project

## 📌 Project Overview

This project demonstrates the implementation of advanced algorithms from Design and Analysis of Algorithms (ADA), including:

* Backtracking Algorithm
* Branch and Bound Algorithm
* String Matching Algorithms
* Performance Analysis using Graphs

The main goal is to solve a **crew scheduling problem**, where flights are assigned to crew members under constraints such as time overlap. The project also analyzes string matching performance for different input sizes.

---

## 🎯 Objectives

* Implement constraint satisfaction using Backtracking
* Optimize solutions using Branch and Bound
* Perform pattern searching using String Matching
* Analyze algorithm performance using graphs
* Understand scalability issues in NP-hard problems

## 🧩 Algorithms Implemented

### 🔹 1. Backtracking

* Recursively assigns flights to crew members
* Ensures no overlapping schedules
* Explores all possible solutions

### 🔹 2. Branch and Bound

* Uses cost function to prune search space
* Reduces unnecessary computations
* Improves efficiency over backtracking

### 🔹 3. String Matching

* Implements Naive pattern searching
* Counts number of comparisons
* Evaluates performance for increasing input size

---

## 📊 Performance Analysis

* Execution time measured for different input sizes
* Graph plotted using matplotlib
* Shows how time increases with input size

## 🧠 Reflection

* Backtracking has exponential time complexity and does not scale well
* Branch and Bound significantly reduces search space
* String matching performance depends on input size
